# Image Locker React App

## Installation Front

Pour lancer l'appli:

1. allez dans le répetoire:
    ```bash
    cd image_locker_react_app
    ```
2. installer les dépendances:
    ```bash
    npm run build:browser
    ```
3. lancer l'appli serverless:
    ```bash
    npx serverless offline start
    ```
